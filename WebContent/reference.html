Login to Google cloud console
gcloud projects list

In bit bucket the project name is m-mailer. Get the url by logging into bitbucket and searching for your project

git clone https://mrireddy@bitbucket.org/jravi123/m-mailer.git

At Google cloud the same project is called flag-alert and this is in mbcc accoumt

To see the current config settings use:
gcloud config list
If the project that is set is not 'flag-alert', then set the project to flag-alert.
The below command sets the project to flag-alert
gcloud config set project flag-alert

Before you build ensure that you are inside the m-mailer project in the console, which has the pom.xml

move to project m-mailer by executing
'cd m-mailer'

Using Maven build the project by using

`mvn clean pakage`

Deploy to app engine (Production - google servers)

`mvn appengine:deploy`

To start a local appengine, run:

`mvn appengine:run`
This will start a local server at localhost:8080
Admin console will be at localhost:8080/_ah/admin
